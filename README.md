## 🌌 О проекте

**Universal** — это idle-кликер о выживании одинокого корабля в недружелюбной галактике. Ваша цель: собирать мусор, расширять отсеки и накопить достаточно мощности для финального гиперпрыжка.

Проект разработан на движке **Godot 4.3** с использованием современных инструментов ИИ для ускорения разработки и обеспечения высокого качества кода.

## ✨ Основные возможности (MVP)

- 🛸 **Строительство корабля:** Расширяйте корпус и устанавливайте функциональные модули.
- ♻️ **Сбор ресурсов:** Собирайте металл вручную или автоматизируйте процесс с помощью специальных сборщиков.
- 📦 **Экономика:** Управляйте лимитами складов и оптимизируйте добычу для достижения максимальной эффективности.
- 🎯 **Система прогрессии:** Улучшайте корабль, пока не достигнете параметров, необходимых для гиперпрыжка.

## 🛠 Технологический стек

- **Движок:** [Godot 4.3](https://godotengine.org/) (GDScript)
- **Архитектура:** Event-Driven (Signaling Bus) для слабой связанности компонентов.
- **Интеграция ИИ:**
  - **Context7:** Актуальная документация и лучшие практики.
  - **Godot-MCP:** Прямое управление сценами и скриптами через LLM.

## 📂 Структура проекта

```text
res://
├── core/           # Глобальные системы (Autoloads, GameEvents)
├── entities/       # Игровые объекты (Ship, Modules, Debris)
├── ui/             # Интерфейс пользователя (HUD, Shop)
├── data/           # Файлы баланса (.tres)
└── assets/         # Звуки, текстуры и шрифты
```

## 🚀 Как запустить

1. Установите **Godot 4.3** или новее.
2. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/vlad/univers.git
   ```
3. Откройте файл `project.godot` в редакторе Godot.
4. Нажмите **F5**, чтобы начать приключение!
